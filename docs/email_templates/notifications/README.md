# Notification Email Templates

Reserved for future general-purpose announcements sent to members or groups.

## Planned Use Cases

- Event reminders and announcements
- Community news bulletins
- Board meeting notices
- Facility maintenance alerts
- General member communications

## No Templates Yet

Templates will be added here as needs arise. When adding a new notification template:

1. Name it using the `NOT_` prefix convention (e.g., `NOT_FACILITY_CLOSURE_TO_MEMBER.txt`)
2. Add the Drive file ID to the Email Templates sheet (GEA System Backend)
3. Add an entry to `_MANIFEST.md` in the parent folder
4. Update this README with the template details
