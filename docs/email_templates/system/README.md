# System Email Templates

Reserved for future automated system and account management emails.

## Planned Use Cases

- Password reset requests
- Account locked notifications
- Scheduled maintenance warnings
- System error alerts to board
- Session expiration notices

## No Templates Yet

Templates will be added here as needs arise. When adding a new system template:

1. Name it using the `SYS_` prefix convention (e.g., `SYS_PASSWORD_RESET_TO_MEMBER.txt`)
2. Add the Drive file ID to the Email Templates sheet (GEA System Backend)
3. Add an entry to `_MANIFEST.md` in the parent folder
4. Update this README with the template details
