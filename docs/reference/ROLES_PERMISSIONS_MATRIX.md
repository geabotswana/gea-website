# GEA Roles & Permissions Matrix

**Reference Document**  
Last Updated: March 3, 2026

---

## Five Primary Roles

1. **Member** – Active GEA member (Full, Associate, Affiliate, Diplomatic, Temporary, Community/Guest)
2. **Treasurer** – Board member; payment verification & financial oversight
3. **Board** – Elected leadership; approval authority
4. **RSO** – Embassy Recreation Services; guest list & facility oversight
5. **Applicant** – Applying for membership (not yet activated)

## Treasurer Permissions

- View payment dashboard (all submissions)
- View household payment history
- Verify payments (reconcile against banks)
- Activate membership after payment
- Suspend facility access (unpaid members)
- Generate payment reports (annual/monthly/category)
- Reconcile bank accounts monthly
- Track exchange rate (BWP/USD)
- Issue receipts (cash payments)
- View member directory
- Communicate payment reminders/invoices
- Attend Board meetings and vote

---

## Board Permissions

**Approvals:**
- Approve/deny membership applications
- Approve Tennis/Basketball excess bookings
- Approve Leobo bookings (2nd stage)
- Approve Whole Facility bookings (2nd stage)
- Approve Affiliate members (case-by-case)

**Oversight:**
- View reservation dashboard
- View household usage statistics
- View member directory & payment info
- Update dues amounts
- Update membership categories
- Modify facility rules
- Set monthly exchange rate
- Communicate approvals/denials/policies
- Attend Board meetings and vote

---

## RSO (Recreation Services) Permissions

- View guest lists (submitted by deadline)
- Approve guest lists (acceptable entries)
- Reject guest lists (incomplete/format issues)
- Reject individual guests (security concerns)
- Request guest list clarification
- View facility reservations
- View final guest lists for facility staff
- Report facility issues (damage, safety, maintenance)
- Manage facility closures (emergency/maintenance)
- Communicate guest list rejections/approvals

**Cannot:**
- Approve/deny reservations (Board authority only)
- Vote on Board decisions

---

## Applicant Permissions

**Can:**
- Complete membership application
- Submit required documents (passport, Omang, photo, employment verification)
- View application status
- Receive email updates
- Submit payment proof (after Board approval & RSO document review)

**Cannot (Until Activated):**
- Book facilities
- Vote or attend GEA events
- Access member directory
- Run for Board

## Communication Requirements

**Treasurer:**
- Payment verification: 2 business days
- Dues invoices: July (30-day) and August 1
- Payment reminders: July/August cycle

**Board:**
- Membership decisions: 3 business days
- Reservation approvals: When stage changes
- Approval reminders: Daily 6:00 AM if pending >1 day

**RSO:**
- Guest list rejections: 5 business days
- Guest list approvals: Upon completion
- Facility issues: To relevant contact

---

## Special Permissions

**Full Members Only:**
- Sponsor applicants for other membership types
- Vote in elections
- Run for Board positions

**Associate Members:**
- Full facility access (same as Full)
- Host guests (if Board approves by facility)

**Temporary Members:**
- Limited facility access (Board discretion)
- No guest hosting (unless Board approves)
